\hypertarget{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md25}{}\doxysubsection{Displaying aggregates only}\label{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md25}
The switch {\ttfamily -\/a} / {\ttfamily -\/-\/display\+\_\+aggregates\+\_\+only} can be used to control the displayment of the normal iterations vs the aggregates. When passed, it will be passthrough to the benchmark binaries to be run, and will be accounted for in the tool itself; only the aggregates will be displayed, but not normal runs. It only affects the display, the separate runs will still be used to calculate the U test.\hypertarget{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md26}{}\doxysubsection{Modes of operation}\label{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md26}

This is a mix of the previous two modes, two (potentially different) benchmark binaries are run, and a different filter is applied to each one. As you can note, the values in {\ttfamily Time} and {\ttfamily C\+PU} columns are calculated as {\ttfamily (new -\/ old) / $\vert$old$\vert$}.\hypertarget{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md27}{}\doxysubsection{U test}\label{md__home_mary_VegaProgramEngineering_2024_build__deps_googlebenchmark-src_docs_tools_autotoc_md27}
If there is a sufficient repetition count of the benchmarks, the tool can do a \href{https://en.wikipedia.org/wiki/Mann\%E2\%80\%93Whitney_U_test}{\texttt{ U Test}}, of the null hypothesis that it is equally likely that a randomly selected value from one sample will be less than or greater than a randomly selected value from a second sample.

If the calculated p-\/value is below this value is lower than the significance level alpha, then the result is said to be statistically significant and the null hypothesis is rejected. Which in other words means that the two benchmarks aren\textquotesingle{}t identical.

{\bfseries{W\+A\+R\+N\+I\+NG}}\+: requires {\bfseries{L\+A\+R\+GE}} (no less than 9) number of repetitions to be meaningful!
